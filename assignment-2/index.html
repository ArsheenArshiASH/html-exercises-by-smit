Web Development Terminologies:

Web Development field is filled with many terms that can be baffling to many.
some of them are mentioned below:

Bug:
A bug is an error that prevents a website or app from running as it should.In the world of web you will often deal with bug reports and be tasked with fixing said bugs.
Deployment:
Deployment is a key step in the web development process.once a website has been develop and tested, it is ready to go live;
in other words, it's ready to be deployed.
Http:
Http Stands for Hypertext Transfer Protocol, and is Used to transfer data across the internet.
JavaScript:
JavaScript is the programming language of the web . It can update and change both HTML and CSS.
It can calculate, manipulate and validate data.
Text Editor:
One Of the key tools you'll use in web development is a text editor. Text editor is use to write plain, unformatted text.
You will use text editors suchas SublimeText, TextEdit or Vscode etc.
